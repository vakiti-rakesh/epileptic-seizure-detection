# 🧠 Epileptic Seizure Detection using Machine Learning

## 📌 Overview

This project focuses on detecting epileptic seizures using machine learning techniques applied to EEG (Electroencephalogram) data. The goal is to build a predictive system that can assist in early diagnosis and monitoring of epilepsy.

---

## 🚀 Features

* 📊 EEG data preprocessing and cleaning
* 🤖 Machine Learning model for seizure detection
* 📈 Data visualization and analysis
* ⚡ Fast prediction system
* 🧪 Model evaluation with accuracy metrics

---

## 🛠️ Tech Stack

* Python
* NumPy, Pandas
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

## 📊 Dataset

* EEG dataset containing brain signal recordings
* Features represent brain activity signals
* Target: Seizure (1) / No Seizure (0)

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/vakiti-rakesh/epileptic-seizure-detection.git
cd epileptic-seizure-detection
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the project

```
python src/main.py
```

## 📈 Model Performance

* Accuracy: 85%
* Precision: 80%
* Recall: 89%
* F1 Score: 81%

## 🔍 How It Works

1. Load EEG dataset
2. Preprocess data (cleaning, normalization)
3. Train ML model (e.g., Logistic Regression / Random Forest)
4. Evaluate performance
5. Predict seizures
